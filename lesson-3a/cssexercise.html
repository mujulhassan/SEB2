Programming Languages for websites

PHP and JavaScript

The PHP and JavaScript programming languages allow make the site dynamic, that is, responsive to user actions. For example, you can make a beautiful drop-down menu or slider.

Types of scripts

For this, scripts are written (English script - "script") - programs that allow you to respond to user actions. Scripts are of two types:

  • those that run on the server, and the result of their execution comes to the browser to ready for the user. These are the scripts written in PHP. Write on it CMS - content management systems.
  • those that run directly in the user's browser. These are scripts written in JavaScript. They are most often used to make a page more comfortable and beautiful.
